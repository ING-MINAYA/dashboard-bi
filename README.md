# Dashboard B.I. Business Intelligence

## Descripción del Proyecto

Este proyecto consiste en una aplicación web desarrollada con React + Vite orientada a la visualización de métricas empresariales mediante un Dashboard de Business Intelligence (B.I.).

El sistema permite mostrar información importante sobre ventas, clientes, inventario, reportes y configuración general del negocio, facilitando la toma de decisiones dentro de una empresa.

El objetivo principal es centralizar la información empresarial en una interfaz clara, moderna y fácil de usar.

## Justificación Técnica

Se utilizó React por su sistema basado en componentes reutilizables, permitiendo una mejor organización y mantenimiento del código.

Vite fue seleccionado por su rapidez en el desarrollo y mejor rendimiento.

React Router DOM permite una navegación fluida entre páginas sin recargar la aplicación.

CSS fue implementado para lograr una interfaz profesional, limpia y orientada a entornos empresariales.
